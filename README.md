# Retail Sales Analysis

A data analysis project using **Python**, **Excel**, and **Git** to uncover retail sales trends by product, region, and month.

---

## Folder Structure
```
Retail-Sales-Analysis/
│
├── data/
│ ├── sales_data.csv
│ └── product_lookup.csv
│
├── images/
│ ├── Sales Monthly Trend.png
│ ├── Revenue - product vs region.png
│ └── Revenue by product category.png
│
├── RETAIL SALES ANALYSIS.ipynb
└── README.md

```


---

## Tools & Libraries

- **Python**: pandas, matplotlib, seaborn
- **Excel**: pivot tables, VLOOKUP
- **Git & GitHub**: version control and hosting

---

## Project Goals

- Analyze sales data by **product**, **region**, and **month**
- Merge product lookup to enrich data
- Visualize trends with charts and pivot tables
- Automate insights with Python + Excel

---

## Key Insights

- Top-selling regions and categories
- Monthly revenue trends
- High-performing products by revenue

---

## Skills Demonstrated

- Data cleaning and transformation (Pandas)
- Data visualization (Matplotlib, Seaborn)
- Business insight generation
- GitHub project structure
